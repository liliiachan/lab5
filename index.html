Контроль корректности имени пользователя

Введите имя:

(Имя должно начинатся с заглавной буквы, остальные символы нижнего регистра;
имя не должно превышать 16 символов)

Контроль корректности адреса электронной почты

Введите адрес эл. почты:

(@ должен быть не первым символом; должна присутствовать минимум 1 точка;
точка должна стоять после @ и должен быть минимум 1 символ между ними;
после точки должны присутствовать минимум 2 символа)

Контроль корректности номера телефона

Введите номер телефона:

(Номер должен содержать не более 12 цифр; номер должен начинаться с 38;
номер не должен состоять только из цифр)
Пример корректного номера: 380123456789
